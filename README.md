# Texas-Holdem
A javafx project for a college class where I created a game of Texas Hold'em. 
There are four players, including the user and 3 computer players.
I included a gamemode with no betting involved, in this case the player needs to click the deck to get to the next stage (flop, turn, etc.). In this gamemode there is no decision-making, and the game ends after a number of rounds predetermined by the player in the menu have passed.
The second and more interesting gamemode is a version of Limit Hold'em, meaning all bets and raises are limited by a predtermined amount (2 chips to raise in this case). The player can determine number of starting chips for everyone in the menu. This gamemode ends when a player has less than 2 chips (cannot post big blind or call).
